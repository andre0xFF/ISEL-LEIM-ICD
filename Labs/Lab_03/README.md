# Item 1

Client request in XML format:

```xml
<p>
    <pedido>
        <op>1</op>
    </pedido>
</p>
```

Server response in XML format:

```xml
<p>
    <pedido>
        <op>1</op>
    </pedido>
    <resposta>
        <valor>...</valor>
    </resposta>
</p>
```

<p>

    </pedido>
